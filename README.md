# Cours 6 - créer une structure de page

Le principe est le même que celui qu’on a mis en œuvre au cours 5 avec le centrage des blocs, sauf qu’on va étendre le raisonnement à plusieurs blocs plutôt que de se limiter à un seul.

## Au programme:
- les balises sémantiques (`header`, `footer`, `nav`, `main`, `section`) + la différence avec `div`
- établir une stratégie de découpe de notre structure
- utiliser le display flex
- utiliser les unités %
- créer un reset des styles de base avec sélecteur universel *
